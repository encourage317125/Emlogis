WFL Test tool

Welcome to the Emlogis scheduler workflow testing tool. This application provides a REST API for a very basic functionality testings. The following paths are supported for POST/GET requests:

  • /rest/calc/add/{left}/{right}
  • /rest/calc/subtract/{left}/{right}
  • /rest/calc/multiply/{left}/{right}
  • /rest/calc/divide/{left}/{right}

The request body must be a JSON object similar to the above ("result" is of course optional). The response will be JSON as above.
